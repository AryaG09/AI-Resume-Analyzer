
# 📄 AI Resume Analyzer & Career Guide

An AI-powered web application that analyzes resumes, extracts technical skills, matches them with job roles, and identifies skill gaps.

Built using Python and Streamlit.

---

##  Features

- Upload resume (PDF)
- Automatic skill extraction
- Job-role based skill matching
- Resume strength scoring
- Skill gap identification
- Modern Purple AI themed dashboard

---

##  Tech Stack

- Python
- Streamlit
- PDFPlumber
- Custom Skill Matching Logic
- Streamlit Theme Configuration

---

##  How It Works

1. Extracts text from uploaded PDF resume  
2. Detects predefined technical skills  
3. Compares with required skills for selected job role  
4. Calculates match percentage  
5. Highlights missing skills  

---

## ▶️ Run Locally

pip install streamlit pdfplumber
streamlit run app.py
